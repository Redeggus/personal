Java Decoder

Created a decoder in Java that takes inputs from a file. For example I decoded a picture of the Mona Lisa First, the decoder gets the encoding scheme and creates a binary tree for decoding. The program then decodes the message in the file using the tree created earlier. As an extra challenge, I created my tree iteratively as opposed to recursively, for some extra credit.
